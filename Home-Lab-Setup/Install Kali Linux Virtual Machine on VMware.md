# Install Kali Linux Virtual Machine on VMware

1. Open the Kali Linux download page:  
   https://www.kali.org/get-kali/#kali-platforms

2. Select **Virtual Machines**.

3. Download the **VMware** version using either:
   - Direct download, or
   - Torrent download.

4. After the download is complete, extract the archive file.

5. Open **VMware Workstation**.

6. Click **Open a Virtual Machine**.

7. Browse to the extracted Kali Linux folder, select the `.vmx` file, then click **Open**.

8. The Kali Linux virtual machine will appear in VMware.

9. Start the virtual machine by clicking **Power on this virtual machine**.

10. Log in using the default credentials:
    - **Username:** kali  
    - **Password:** kali

---

✅ Kali Linux virtual machine is successfully installed and ready to use.
