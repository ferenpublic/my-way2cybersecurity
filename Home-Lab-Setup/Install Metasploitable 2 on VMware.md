# Install Metasploitable 2 on VMware

1. Open the Metasploitable 2 download page:  
   https://www.vulnhub.com/entry/metasploitable-2,29/

2. Click the **Download** link.

3. After the download is complete, extract the archive file.

4. Open **VMware Workstation**.

5. Click **Open a Virtual Machine**.

6. Browse to the extracted folder, select **Metasploitable.vmx**, then click **Open**.

7. Start the virtual machine by clicking **Power on this virtual machine**.

8. Log in using the default credentials:
   - **Username:** msfadmin  
   - **Password:** msfadmin

---

✅ Metasploitable 2 virtual machine is successfully installed and ready to use.
