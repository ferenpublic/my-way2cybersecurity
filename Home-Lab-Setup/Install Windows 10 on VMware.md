# Install Windows 10 on VMware

1. Obtain a **Windows 10 ISO** file  
   *(Example used: Windows 10 Multiple Editions English 64-bit)*.

2. Open **VMware Workstation** and click **Create a New Virtual Machine**.

3. Select **Typical (recommended)**, then click **Next**.

4. Choose **Installer disc image file (ISO)**, browse to the Windows 10 ISO file, then click **Next**.

5. Configuration:
   - Leave **Windows Product Key** empty
   - Select **Windows 10 Home Single Language** as the version
   - Enter full name and optional password as preferred  
   Click **Next**.

6. When prompted with **You did not enter a Windows product key**, click **Yes**.

7. Set the **Virtual Machine Name** and storage location, then click **Next**.

8. Configure disk settings:
   - Disk size: **32 GB** (minimum recommended)
   - Select **Store virtual disk as a single file** or preferred option  
   Click **Next**.

9. Click **Finish**.

10. Start the virtual machine by clicking **Power on this virtual machine**.

11. Select **Windows 10 Home**, then click **Next**.

12. Wait for the installation process to complete.

---

✅ Windows 10 virtual machine installation completed successfully.
