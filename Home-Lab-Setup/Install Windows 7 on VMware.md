# Install Windows 7 on VMware

1. Obtain a **Windows 7 ISO** file  
   *(Example used: Windows 7 Ultimate 64-bit)*.

2. Open **VMware Workstation** and click **Create a New Virtual Machine**.

3. Select **Typical (recommended)**, then click **Next**.

4. Choose **Installer disc image file (ISO)**, browse to the Windows 7 ISO file, then click **Next**.

5. Configuration:
   - Leave **Windows Product Key** empty
   - Select **Windows 7 Ultimate** as the version
   - Enter full name and password as preferred  
   Click **Next**.

6. When prompted with **You did not enter a Windows product key**, click **Yes**.

7. Set the **Virtual Machine Name** and storage location, then click **Next**.

8. Configure disk settings:
   - Disk size: **20 GB** (minimum recommended)
   - Select **Store virtual disk as a single file** or preferred option  
   Click **Next**.

9. Click **Finish**.

10. Start the virtual machine by clicking **Power on this virtual machine**.

11. Wait for the Windows installation process to complete.

12. Select **Home network** when prompted.

13. Keep default Homegroup settings, then click **Next**.

14. Save the configured password securely, then click **Finish**.

---

✅ Windows 7 virtual machine installation completed successfully.
