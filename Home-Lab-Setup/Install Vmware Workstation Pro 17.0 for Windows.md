# Install VMware Workstation Pro 17.0 for Windows

1. Open the Broadcom Support Portal:  
   https://support.broadcom.com

2. Log in or register an account.

3. After successful login, access the VMware Workstation Pro downloads page:  
   [**Download VMware Workstation Pro (Free)**](https://support.broadcom.com/group/ecx/productdownloads?subfamily=VMware%20Workstation%20Pro&freeDownloads=true)

4. Select **VMware Workstation Pro 17.0 for Windows**.

5. Expand the version list and choose the latest version **17.6.4** for 17.0 edition.

6. Click **Download**.

7. If additional verification is requested:
   - Click **Yes**
   - Complete the verification process
   - Click **Download** again

8. After the download is complete, run the `.exe` installer file.

9. Installation steps:
   - Click **Next**
   - Select **I accept the terms in the License Agreement**, then click **Next**
   - Click **Next**
   - Change the destination folder if needed or keep the default location  
     Enable **Add VMware Workstation to PATH**, then click **Next**
   - Configure optional settings as preferred, then click **Next**
   - Configure shortcut options as preferred, then click **Next**
   - Click **Install**

10. After installation finishes, click **Finish**.

---

✅ VMware Workstation Pro is now installed and ready to use.
